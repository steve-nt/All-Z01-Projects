# Tetris Optimizer

## Description

The program reads a file containing one or more tetrominoes and arranges them (without rotating) in the smallest possible square.
The output is printed on the terminal, with each tetromino having been assigned a different letter of the alphabet.

## Input format

Tetris Optimizer takes as input a txt file containing one or more tetrominoes in the following format:

`....`   
`.##.`    
`##..`    
`....`

Each tetromino must be in a 4x4 grid of points. The tetromino itself is marked by four interconnected #.
In case of more than one tetrominoes, they must be separated by an empty line.

## How to run

Run the command `go run . <path to input file>`
