-- Usage --
1. git clone https://platform.zone01.gr/git/apapatha/go-reloaded.git
2. To start the program, run the following command and specify the output file name:
    go run . sample.txt result.txt 
3. You will be prompted to select a scenario
4. After select a scenario run the following command
    cat result.txt
   and look if the output match
5. Do it for any scenario if you want
