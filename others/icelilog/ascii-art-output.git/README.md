# Welcome to the Whimsical ASCII Art Adventure! 🎨✨

Hello, wonderful code wizard! 🧙‍♀️ Are you ready to turn your plain text into marvelous ASCII masterpieces? You've just stumbled upon a delightful little package that’s here to sprinkle some joy on your strings. Let's dive into the fun!

## 🚀 How to Get Started:

Running this code is as easy as waving a magic wand! Just use the following incantation in your terminal:

```bash
go run . [OPTIONS] [STRING] [BANNER]
```


### 🌈 Options for Extra Fun:
- `--align=`: Decide how your art aligns on the page! (left, right, center, or justify)
- `--output=`: Save your enchanting creations to a file for safekeeping!


### 🎉 Examples of Awesomeness:

```bash
go run . "Hello, World!" standard
```
This spell will transform "Hello, World!" into a stunning ASCII art banner using the "standard" style. Ta-da! 🎊

```bash
go run . --align=right "Go is Awesome!" thinkertoy
```
This incantation aligns "Go is Awesome!" to the right using the "thinkertoy" ASCII style. Perfect for a fun, laid-back vibe! 🎈

```bash
go run . --align=center "Code Like a Pro!" shadow
```
This command centers "Code Like a Pro!" with a shadow-y flair, making it the star of the show! 🌟

```bash
go run . --output=art_output.txt "ASCII Art Rocks!" standard
```
With this spell, "ASCII Art Rocks!" is saved to a file called art_output.txt using the "standard" style, so you can keep it forever! 📄💖



## ✨ What Magical Powers Does This Have?

1. **Align Your Art**: Choose how your text dances across the screen:
   - **Left**: Classic and classy, like a perfectly brewed cup of tea.
   - **Right**: For when you want to be a little fancy-pants!
   - **Center**: Show off your art with a dash of elegance.
   - **Justify**: Neatly spaced, just like a well-organized sock drawer!

2. **Output to File**: Want to keep your creative genius forever? Save it to a file and cherish it like a rare treasure! 🏴‍☠️💎

## 🎭 Meet the Fun Functions Inside:

This package is packed with quirky little helpers that make everything come to life:

- **PrintAsciiArt**: The magical function that transforms characters into their dazzling ASCII forms! 🎨✨
- **PrintAsciiArtToFile**: Capturing your creations, like a photographer at a wedding!
- **PrintAsciiArtAlign**: Aligning your text so it struts its stuff, no matter the occasion!
- **GetTerminalSize**: This nifty function checks the size of your terminal! It retrieves the width and height, ensuring your ASCII art fits perfectly, like a well-tailored suit! 🧵✨
