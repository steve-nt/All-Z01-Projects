## Description
This project is introducing us to the world of version control and collaboration using Git and demonstrates the use of bash scripts with Git version control. It includes:

- A `Greeter` function in `lib/greeter.sh`.
- A script `lib/hello.sh` to greet users with customizable input.
- A `Makefile` to simplify running the script.

## How to Run
1. Ensure `bash` is installed.
2. Run the command: `make run`.

## Branches
- `main`: Stable version.
- `greet`: Adds greeting functionality and updates the script structure.
