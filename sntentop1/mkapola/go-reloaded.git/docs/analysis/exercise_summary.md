# Very briefly...

We will create a **tool** that will process plain text. It will take two files: one containing the text to be processed and another empty one where the processed text will be saved. So, for the program to work, we will **input the names of the two files**, and it will **output the previously empty file now containing the corrected text** (see more detailed installation instructions in the file run_program.md).

The original text will contain **parentheses with one or two words inside**, which will indicate **what kind of transformation** needs to be done and **on which words**. The program will **remove the parentheses** along with their contents when they contain commands.
