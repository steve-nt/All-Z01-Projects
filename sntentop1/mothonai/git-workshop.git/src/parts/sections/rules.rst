#####
Rules
#####

There is a set of rules one needs to follow, especially for projects based on
collaborative work.


***********
Contributor
***********

* Keep branch updated and clean.
* Make small, clear commits.
* Follow coding guidelines.
* Test before PR.
* Give context and respond to feedback.

**********
Maintainer
**********

* Review PRs carefully.
* Keep main branch stable.
* Merge only when standards are met.
* Manage issues and labels.
* Guide contributors consistently.
