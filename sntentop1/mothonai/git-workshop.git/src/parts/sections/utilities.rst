``git stash``
*************

-  ``git stash`` and ``git stash pop``

**********************
``git commit --amend``
----------------------

Amends the latest commit with more information.

********************
``git reset --soft``
********************

Full syntax:

``git reset --soft <commit-hash>``

Resets the current HEAD at given commit and keeps changes since then into the
working tree.

********************
``git reset --hard``
********************

Full syntax:

``git reset --hard <commit-hash>``

Resets the current HEAD at given commit and discards changes since then.
