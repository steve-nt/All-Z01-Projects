This subject introduces **networking and security basics** every developer should understand.

You are not expected to become a network engineer,  
but you must understand how computers communicate.

### Objectives

You must:

- Understand basic networking concepts
- Learn basic cybersecurity notions
- Validate a **Cisco Networking Academy certificate**

### Requirements

You must complete **ONE** Cisco Networking Academy course, such as:

- [Networking Basics](https://www.netacad.com/courses/networking-basics?courseLang=en-US)
- [Introduction to Cybersecurity](https://www.netacad.com/courses/introduction-to-cybersecurity?courseLang=en-US)

### Deliverables

- A Cisco certificate of completion
