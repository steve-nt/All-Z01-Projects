This project introduces you to **cloud fundamentals** using AWS learning resources.

As a developer, you write code, but this code always runs somewhere.  
The goal of this subject is to understand **where and how your applications live in the cloud**.

### Objectives

By completing this subject, you must:

- Understand basic cloud concepts (compute, storage, network)
- Understand the shared responsibility model
- Be able to explain how a web application could be hosted on AWS
- Validate an official **AWS Digital Badge**

### Requirements

You must complete **ONE** of the following AWS badges:

- [AWS Cloud Quest: Cloud Practitioner](https://skillbuilder.aws/learn/FU5WCYVGKY/aws-cloud-quest-cloud-practitioner/JF9TKU68GT) ~12h
- [AWS Knowledge: Cloud Essentials](https://skillbuilder.aws/learning-plan/XKS3J4FAS7/cloud-essentials--knowledge-badge-readiness-path--includes-labs/Y4YASRJEVX) ~22h

### Deliverables

- A valid **AWS badge** (Credly link required)
