Being a professional developer requires more than knowing a programming language.
It also requires solid foundations in computer science, problem solving, and
software logic.

In this project, you will complete **two mandatory academic certifications**
from prestigious universities via the edX platform, in order to validate your
core Computer Science knowledge.

You may also complete **additional optional certifications** if you wish to go
further, but they are not required to validate this project.

### Mandatory Certifications

You **must complete both** of the following courses:

- **CS50’s Introduction to Computer Science** – Harvard University  
  https://www.edx.org/learn/computer-science/harvard-university-cs50-s-introduction-to-computer-science

- **CSE101x: Introduction to Computer Science** – University of Tokyo  
  https://learning.edx.org/course/course-v1:ScienceTokyo+CSE101x+1T2025/home

These two courses are mandatory and provide complementary approaches to
computer science fundamentals from internationally recognized institutions.

### Optional Certifications (Not Required)

If you want to deepen your knowledge, edX offers many high-quality courses
from prestigious universities and tech companies.  
Here are a few examples:

- **CS50’s Introduction to Programming with Python** – Harvard University  
  https://www.edx.org/learn/python/harvard-university-cs50-s-introduction-to-programming-with-python

- **CS50’s Introduction to Artificial Intelligence with Python** – Harvard University  
  https://www.edx.org/learn/artificial-intelligence/harvard-university-cs50-s-introduction-to-artificial-intelligence-with-python

- **Python Basics for Data Science** – IBM  
  https://www.edx.org/learn/python/ibm-python-basics-for-data-science

Learners are encouraged to explore other Computer Science courses available on:
https://www.edx.org/search?attributes=Computer+Science&productType=Course

### Expected Skills

This project will help you strengthen:

- Core computer science concepts
- Algorithms and problem solving
- Programming fundamentals
- Code logic and structure
- Academic rigor and technical autonomy

### Deliverables

- Completion of the **two mandatory edX courses**
- Proof of completion (certificates or official edX pages)
