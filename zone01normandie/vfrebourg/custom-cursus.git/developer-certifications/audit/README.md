#### Functional

##### Verify certification completion.

###### Has the learner completed at least 3 mandatory FreeCodeCamp certifications?

###### Are the certifications part of the approved mandatory list?

##### Check project validation.

###### Are all 5 projects for each certification marked as completed on FreeCodeCamp?

###### Is the learner’s FreeCodeCamp profile public and accessible?

##### Knowledge validation.

###### Can the learner explain concepts related to the completed certifications? (e.g. HTML/CSS layout, JavaScript logic, REST APIs, databases, testing)

###### Can the learner explain how these skills apply to real-world development projects?

#### Bonus

###### + Has the learner completed more than 3 certifications?

###### + Has the learner completed Quality Assurance or Back End certifications?

###### + Does the learner show consistency and autonomy across projects?
