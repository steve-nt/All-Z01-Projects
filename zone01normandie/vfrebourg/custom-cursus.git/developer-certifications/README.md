FreeCodeCamp is a project-based learning platform focused on practical web
development skills. Each certification is earned by completing a series of
hands-on projects that must pass automated and functional tests.

In this project, you will complete **several FreeCodeCamp certifications**
selected to strengthen your core developer skills in front-end, back-end,
databases, and software quality.

### Mandatory Certifications

You must complete **at least 3** of the following certifications:

- **Responsive Web Design**
  - HTML
  - CSS
  - Accessibility and layout fundamentals

- **JavaScript Algorithms and Data Structures**
  - Programming fundamentals
  - Logic, loops, conditionals
  - Basic algorithms and data manipulation

- **Relational Database**
  - SQL (PostgreSQL)
  - Database design
  - Bash, Git, and CLI basics

- **Back End Development and APIs**
  - Node.js
  - Express
  - REST APIs
  - MongoDB

- **Quality Assurance**
  - Functional and unit testing
  - Chai
  - Advanced backend logic

Each certification includes **5 real-world projects** that must be completed
and validated on the FreeCodeCamp platform.

### Optional Certifications (Not Required)

Learners who want to specialize further may explore additional FreeCodeCamp
certifications (Front-End Libraries, Data Visualization, Python, Security, etc.),
but these are not required to validate this project.

### Expected Skills

This project will help you develop:

- Solid web development fundamentals
- Problem solving and programming logic
- Front-end and back-end understanding
- Database design basics
- Code quality and testing mindset
- Autonomy and technical rigor

### Deliverables

- Completion of **at least 3 mandatory FreeCodeCamp certifications**
- Public FreeCodeCamp profile link
- Proof that all required projects are validated
