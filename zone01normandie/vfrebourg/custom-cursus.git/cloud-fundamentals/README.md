This subject introduces **modern cloud concepts** using Google Cloud learning resources.

Google Cloud focuses on **containers, data and scalable applications**.  
The goal is to understand how modern applications are designed and deployed.

### Objectives

You must:

- Understand container-based applications
- Understand basic cloud services and managed solutions
- Discover Kubernetes and its purpose
- Validate an official **Google Skill Badge**

### Requirements

You must complete **ONE** Google Cloud Skill Badge, such as:

- [Cloud Computing Fundamentals](https://www.skills.google/course_templates/153)
- [Kubernetes or Containers Basics](https://www.skills.google/course_templates/744)
- [Data or BigQuery Fundamentals](https://www.skills.google/course_templates/680)

### Deliverables

- A valid Google Skill Badge
