This subject focuses on **enterprise cloud and cybersecurity fundamentals**.

As a developer, security is not optional.  
This subject aims to build **security awareness**, not hacking skills.

### Objectives

You must:

- Understand basic cybersecurity concepts
- Understand common risks in web applications
- Learn how companies approach cloud and security
- Validate an **IBM SkillsBuild badge**

### Requirements

You must complete **ONE** IBM SkillsBuild badge, such as:

- [Cybersecurity Fundamentals](https://students.yourlearning.ibm.com/activity/PLAN-805005E992EA)
- [Cloud Computing Fundamentals](https://students.yourlearning.ibm.com/activity/PLAN-58FA14F64C9B)
- [Information Technology Fundamentals](https://students.yourlearning.ibm.com/activity/PLAN-189E4B1AF551)

### Deliverables

- A valid IBM badge (Credly)
