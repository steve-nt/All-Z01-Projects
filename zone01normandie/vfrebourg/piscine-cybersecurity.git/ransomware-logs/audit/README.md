## host-isolate

#### Functional

###### Is the file present and properly named?

##### Review the structure of the file.

###### Does it include the 4 expected sections: Theory, Techniques, Commands and Tools, Simulation/Case Study?

###### In the Theory section, does the student clearly explain why and when host isolation is necessary?

###### In the Techniques section, are there at least two methods described (Windows and Linux), with optional mention of EDR/network-based methods?

###### +Is there a comparison between manual vs automated isolation?

###### +Is the document well structured and technically accurate?
