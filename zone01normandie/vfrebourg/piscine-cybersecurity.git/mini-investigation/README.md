### find-target

#### Goal

Conduct OSINT investigation on a fictional individual.

#### Instructions

Use provided fake identity data (or generate your own using https://www.fakenamegenerator.com/) to simulate a basic investigation. Use available open-source tools and platforms (e.g., search engines, social media, username search, etc.).

#### Deliverables

- A short document listing public data found (no real PII).
- Mention which tools/methods were used.

### data-timeline

#### Goal

Chronologically organize OSINT findings from a fictional investigation.

#### Instructions

Take the collected data from a previous investigation (e.g., `find-target`) and organize it in a time-based sequence (e.g., spreadsheet, text, markdown table).

#### Deliverables

- A structured timeline with events, dates, and sources.

### cross-ref

#### Goal

Cross-check OSINT sources to validate findings.

#### Instructions

Choose 3–5 data points from the previous investigation and verify them using at least 2 independent sources per point.

#### Deliverables

- A small report explaining how each data point was confirmed.
- Mention tools/platforms used for validation.

### report-out

#### Goal

Draft a brief OSINT investigation report.

#### Instructions

Using the data gathered and verified, write a 1–2 page report simulating a real OSINT output. Focus on clarity, structure, and objective tone. You may redact sensitive or unrealistic details.

#### Deliverables

- The investigation summary (PDF or Markdown).
