### gdpr-terms

#### Goal

Define key GDPR-related terms accurately.

#### Instructions

Research and define the following key terms using your own words:

- Personal data  
- Processing  
- Data subject  
- Data controller  
- Data processor  

Use reliable sources such as the GDPR itself or official summaries (e.g., CNIL, EDPS, or EDPB).

#### Deliverables

- A Markdown file or short PDF with your definitions (max 1 page).

### rights-check

#### Goal

Identify the main rights of a data subject under the GDPR.

#### Instructions

List and explain the rights granted to individuals under GDPR. For each one, briefly describe:

- What the right allows
- A real or hypothetical example

#### Deliverables

- A document listing and explaining each right (max 2 pages).

### legal-bases

#### Instructions
