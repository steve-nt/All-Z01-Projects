## web-pwn

### Goal

Solve a web exploitation challenge focusing on common vulnerabilities.

### Instructions

- Analyze the provided vulnerable web application code snippet
- Identify and exploit at least one vulnerability (e.g., XSS, SQLi, command injection)
- Provide a step-by-step explanation of your exploitation process

### Deliverables

- A write-up explaining the vulnerability
- Exploit code or proof of concept
