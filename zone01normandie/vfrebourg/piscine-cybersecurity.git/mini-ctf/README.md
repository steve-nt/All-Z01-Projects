### web-pwn

#### Goal

Practice exploiting a basic web vulnerability using an online CTF platform.

#### Instructions

1. Choose a beginner-level web challenge from one of the following:
   - https://tryhackme.com/
   - https://www.hackthebox.com/
   - https://picoctf.org/

2. Focus on a challenge involving:
   - XSS
   - SQLi
   - Insecure login
   - IDOR

3. Solve the challenge and document your process.

#### Deliverables

- A short write-up of your exploit steps.
- A screenshot of the final flag or success screen.

### crypto-fun

#### Goal

Solve a basic cryptography puzzle using an online CTF platform.

#### Instructions

1. Choose a beginner cryptography challenge from:
   - https://cryptohack.org/
   - https://picoctf.org/
   - https://ctflearn.com/

2. Focus on classical or beginner crypto tasks like:
   - Caesar cipher
   - XOR encoding
   - RSA basics

3. Solve the task and explain your approach.

#### Deliverables

- A brief explanation of the cryptographic method.
- Your decoding steps or script.
- The final flag or answer.

### file-trace

#### Goal

Analyze file metadata to uncover hidden information using a real file from an external platform.

#### Instructions

1. Download and solve a file analysis challenge from:
   - https://www.hackthebox.com/
   - https://picoctf.org/
   - https://ctflearn.com/

2. Focus on extracting:
   - EXIF metadata
   - Hidden data in images/docs
   - File structure anomalies

3. Document your findings and tools used.

#### Deliverables

- A summary of your analysis.
- Tools or commands used.
- Screenshot or proof of flag found.

### dataset-clues

#### Goal

Use an open dataset to extract and correlate useful information.

#### Instructions

1. Use an open dataset platform such as:
   - https://www.kaggle.com/
   - https://data.gov/
   - https://opendata.paris.fr/

2. Pick a dataset that could contain investigative clues (e.g., leaks, security events, company info).

3. Analyze it using tools like pandas, grep, or spreadsheet filters to find useful clues or patterns.

#### Deliverables

- Description of the dataset used.
- Your analysis steps and findings.
- Optional: script or code used to parse data.
