### owasp-match

#### Goal

Understand how common vulnerabilities map to the OWASP Top 10.

#### Instructions

- Review the OWASP Top 10 list (latest version: https://owasp.org/www-project-top-ten/)
- Match at least 5 common vulnerabilities (e.g. SQLi, XSS, broken access control) to the corresponding OWASP category
- Justify each match with a real or hypothetical example

#### Deliverables

- A markdown table with mappings and explanations
- References to the OWASP list

### owasp-quiz

#### Goal

Validate knowledge of OWASP Top 10 risks through a custom quiz.

#### Instructions

- Create a quiz of 10 questions covering the OWASP Top 10
- Include multiple-choice questions with one or more correct answers
- Write explanations for each correct answer

#### Deliverables

- Quiz file (markdown or PDF)
- Answer key with explanations

### real-case

#### Goal

Analyze a real-world incident in the context of OWASP Top 10.

#### Instructions

- Choose a known cybersecurity breach (e.g., Equifax, Uber 2016)
- Identify which OWASP Top 10 risks were involved or ignored
- Describe the timeline and root causes

#### Deliverables

- Summary report (1–2 pages)
- OWASP risks highlighted
- Lessons learned
