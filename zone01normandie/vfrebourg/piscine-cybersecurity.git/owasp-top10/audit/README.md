## owasp-quiz – Audit

### Functional

##### Does the quiz contain 10 OWASP-related questions?

## real-case

## real-case – Audit

### Functional

##### Did the student analyze a real cyber attack?

###### Did they link it to specific OWASP Top 10 risks?

###### Did they describe the breach clearly?

### Bonus

###### + Root cause analysis

###### + Lessons applicable to prevention
