### Extracurricular Project: Git Workshop

#### Description
The Git Workshop focuses on essential Git concepts and practices to strengthen version control skills among our learners. Participants learn about effective branching strategies, meaningful commit practices, and the importance of clear project documentation with README files. This workshop aims to cultivate disciplined and collaborative development habits, crucial for professional software engineering.

---

### What is expected?

##### General Information

###### Does the project include a README file explaining its purpose, installation, and usage?
###### Are meaningful commit messages used instead of generic ones (e.g., "fix" or "update")?

---

##### Version Control Best Practices

###### Are branches used properly to separate features, fixes, and experiments?
###### Does the project history show well-structured commits reflecting logical changes?
###### Can the participant demonstrate how they managed conflicts and merged branches?

---

##### Project Presentation

###### Can the participant showcase a recent project and explain their Git workflow?
###### Is the GitHub repository structured clearly with appropriate documentation?
###### Does the participant understand the importance of Git in professional development?
