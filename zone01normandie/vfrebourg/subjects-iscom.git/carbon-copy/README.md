# Superhero Agency

## Context
Welcome to **Superhero Agency**, a website dedicated to recruiting the best **super communicators**!

Your mission: develop a website for our prestigious agency that identifies and trains top communication talents. Your website should be attractive, functional, and interactive to attract the ideal recruits.

As a bonus, you will implement a **Excuse Generator**, a fun JavaScript tool that creates hilarious excuses for various situations.

## Learning Objectives
- Develop a complete website using HTML, CSS, and JavaScript.
- Deepen DOM manipulation and dynamic content management.
- Enhance knowledge of communication-related professions.
- Stimulate creativity and boldness through original features.

## Instructions
You must design and develop a website containing the following elements:

### 1. Homepage
- Introduction to the agency with a catchy design.
- A compelling slogan to encourage joining the agency.
- Highlighting the agency's values and mission.

### 2. "Super Communicators" Page
- A list of key skills required to be a super communicator.
- Presentation of different communication-related professions (e.g., Community Manager, Public Relations Officer, Copywriter, etc.).
- A dynamic and engaging layout with CSS or JS animations.

### 3. "Join the Agency" Page
- A registration form with relevant fields (name, email, skills, motivation, etc.).
- JavaScript-based form validation.
- A confirmation message upon submission.

### 4. "Excuse Generator" Page (Bonus)
- An interactive JavaScript section that generates random excuses for various situations.
- Multiple categories (e.g., Late for work, Canceling a meeting, Homework not done, etc.).
- A button to generate a new random excuse.
- An option to share the generated excuse on social media.

## Technical Requirements
- The website must be responsive and accessible on both mobile and desktop.
- Use HTML5, CSS3, and pure JavaScript (no frameworks).
- Well-structured and commented code.
- The design should be modern and attractive.
- Option to use CSS animations to enhance user experience.

## Evaluation Criteria
- Adherence to the specifications and required functionalities.
- Quality of design and website ergonomics.
- Good code structuring and readability.
- Originality and relevance of the content.
- Functionality of the excuse generator.
