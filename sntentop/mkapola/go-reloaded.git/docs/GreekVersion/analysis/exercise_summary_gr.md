## Πολύ συνοπτικά...

Θα φτιάξουμε ένα **εργαλείο** το οποίο θα **επεξεργάζεται απλό κείμενο**. Θα παίρνει δύο αρχεία, το ένα θα έχει μέσα το κείμενο που θα επεξεργαστούμε και το άλλο θα είναι κενό για να μπει μέσα το επεξεργασμένο κείμενο. Οπότε για να λειτουργήσει το πρόγραμμα θα του **βάζουμε σαν input το όνομα των δύο αρχείων** και θα **βγάζει (output) το πρώην κενό αρχείο που τώρα θα έχει το διορθωμένο κείμενο** (δες πιο αναλυτικά τις οδηγίες εγκατάστασης στο αρχείο run_program.md)

Το αρχικό κείμενο θα έχει σε διάφορα σημεία κάποιες **παρανθέσεις με μία ή δύο λέξεις μέσα**, οι οποίες θα δηλώνουν **τι μετατροπή** χρειάζεται να γίνει και **σε ποιες λέξεις**. Θα **διαγράφει τις παρενθέσεις** με το περιεχόμενό τους, όταν περιλαμβάνουν εντολές.
