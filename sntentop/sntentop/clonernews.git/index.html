What this final deliverable includes

clonernews is a polished static Hacker News client with clear architectural boundaries, expressive UI foundations, and a strict safety posture. The app is organized by domain intent: pure core use-cases, infrastructure adapters for I/O, shared primitives, and feature-local views/controllers.

  • Editorial dark visual direction with typography-first design tokens.
  • Hash-based routing for static hosting compatibility.
  • HN API adapter, cache adapter, throttling, and core use-cases are integrated.
  • Feed, post detail, comments, live updates, and audit-friendly test hooks.
  • Live-data notifications are throttled at a 5-second minimum interval.
  • Batch item fetching uses bounded concurrency to keep UI responsive.
  • Network requests use AbortController timeout boundaries.
  • DOM rendering uses safe sinks and sanitized API HTML fields.
  • Vitest, Playwright, and audit traceability map quality to requirements.
  • Static build is deployment-ready for GitHub Pages hosting.
