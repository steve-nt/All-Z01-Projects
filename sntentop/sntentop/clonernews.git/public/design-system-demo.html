Acceptance Criteria

  • ✓ Dark mode primary palette with warm accent (#ffb45c) on navy
  • ✓ Spacing scale with 0.25rem (4px) base unit (7 levels: xs through 3xl)
  • ✓ Google Font pairing: Fraunces (display) + Manrope (body) — distinctive, not Inter
  • ✓ CSS custom properties for all tokens (colors, spacing, radius, shadows)
  • ✓ Smooth micro-animation keyframes: fade-in-up, slide-down, fade-in-left, shimmer, pulse
  • ✓ Global reset: box-sizing, margin reset, font-smoothing
  • ✓ No !important directives in entire stylesheet
  • ✓ Light mode support via prefers-color-scheme media query
  • ✓ Design tokens render correctly on this test page
