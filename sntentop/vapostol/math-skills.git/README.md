# Statistical Calculator

A simple command-line tool written in Go for calculating basic statistical measures—average (mean), median, variance, and standard deviation—from a list of numbers provided in a text file.

## Features

- **Average (Mean):** Calculates the average value of the numbers.
- **Median:** Finds the middle value when the numbers are sorted.
- **Variance:** Measures how spread out the numbers are from the mean.
- **Standard Deviation:** The square root of the variance, indicating the dispersion of the dataset.
